# Chapter 2: Predicate Logic — Quantification and Structure

## 2.1 Introduction: Beyond Propositions

Propositional logic, while powerful in its domain, cannot represent the internal structure of propositions. It cannot capture relationships between objects, make claims about all or some members of a domain, or express statements involving properties that apply to multiple entities. Predicate logic (also called first-order logic) extends propositional logic by introducing quantifiers and predicates, enabling the formal representation of substantially richer logical relationships.

This chapter develops the syntax and semantics of first-order logic, the theory of quantification, and the methods of formal proof appropriate to the predicate calculus.

## 2.2 The Language of First-Order Logic

### 2.2.1 Primitive Symbols

The alphabet of first-order logic includes:

**Logical Connectives:** ¬ (negation), ∧ (conjunction), ∨ (disjunction), → (conditional), ↔ (biconditional)—identical to propositional logic.

**Quantifiers:** ∀ (universal quantifier, "for all"), ∃ (existential quantifier, "there exists").

**Variables:** A countable set of individual variables: x, y, z, x₁, y₁, z₁, ...

**Predicate Symbols:** Symbols that represent properties of objects or relations between objects. An n-place predicate takes n arguments. We use uppercase letters: P, Q, R, ... to denote predicates.

**Function Symbols:** Symbols that denote functions from the domain to the domain. An n-place function symbol takes n arguments and returns a single value. We use lowercase italicized names: f, g, h, ... to denote functions.

**Constant Symbols:** Symbols that denote particular individuals in the domain. We use lowercase letters: a, b, c, ... to denote constants.

**Equality Symbol:** The binary relation =, expressing identity between terms.

### 2.2.2 Terms and Formulas

A **term** is defined recursively:
1. Variables and constants are terms.
2. If f is an n-place function symbol and t₁, ..., tₙ are terms, then f(t₁, ..., tₙ) is a term.
3. Nothing else is a term.

An **atomic formula** is formed by applying a predicate to appropriate terms:
- If P is an n-place predicate and t₁, ..., tₙ are terms, then P(t₁, ..., tₙ) is an atomic formula.
- If t₁ and t₂ are terms, then t₁ = t₂ is an atomic formula.

A **well-formed formula** (wff) is defined recursively:
1. Every atomic formula is a wff.
2. If α is a wff, then ¬α is a wff.
3. If α and β are wffs, then (α ∧ β), (α ∨ β), (α → β), and (α ↔ β) are wffs.
4. If α is a wff and x is a variable, then ∀x α and ∃x α are wffs.
5. Nothing else is a wff.

### 2.2.3 Free and Bound Variables

A variable occurrence is **bound** if it falls within the scope of a quantifier that quantifies that variable. A variable is **free** in a formula if it is not bound by any quantifier in the formula.

**Example:** In the formula ∀x (P(x) → ∃y R(x, y)), both x and y are bound. In the formula P(x) ∧ ∃y Q(y, z), x is free and y is bound; z is free.

A formula with no free variables is called a **sentence** (or closed formula). Only sentences have definite truth values in a given interpretation.

## 2.3 Semantics of First-Order Logic

### 2.3.1 Interpretations

An interpretation I for a first-order language consists of:
1. A non-empty domain D (the universe of discourse).
2. An assignment of an element of D to each constant symbol.
3. An assignment of an n-place relation on D to each n-place predicate symbol.
4. An assignment of an n-place function from Dⁿ to D to each n-place function symbol.

### 2.3.2 Truth Conditions

The truth of a formula in an interpretation is defined recursively:

**Atomic Formulas:** P(t₁, ..., tₙ) is true in I if and only if the n-tuple of objects denoted by t₁, ..., tₙ in I falls under the relation assigned to P in I.

**Negation:** ¬α is true in I if and only if α is false in I.

**Conjunction:** α ∧ β is true in I if and only if α is true in I and β is true in I.

**Disjunction:** α ∨ β is true in I if and only if α is true in I or β is true in I (inclusive).

**Conditional:** α → β is true in I if and only if α is false in I or β is true in I.

**Universal Quantification:** ∀x α is true in I if and only if α is true in every interpretation I' that is identical to I except possibly in the value assigned to the variable x (i.e., for every element d in D, α holds when x is assigned to d).

**Existential Quantification:** ∃x α is true in I if and only if there exists some interpretation I' that is identical to I except possibly in the value assigned to the variable x such that α is true in I' (i.e., there exists some element d in D such that α holds when x is assigned to d).

### 2.3.3 Logical Consequence

A formula α is a **logical consequence** of a set of formulas Γ (written Γ ⊨ α) if and only if every interpretation that makes all formulas in Γ true also makes α true.

A formula is **valid** (logically true) if it is true in every interpretation. A formula is **satisfiable** if there exists some interpretation that makes it true. A formula is **unsatisfiable** (or inconsistent) if there is no interpretation that makes it true.

## 2.4 Quantifier Logic: Key Laws and Transformations

### 2.4.1 Negation of Quantifiers

The following equivalences govern the interaction between negation and quantification:

**Negation of Universal Quantifier:**
¬∀x φ ≡ ∃x ¬φ

The statement "not everything is φ" is equivalent to "there exists something that is not φ."

**Negation of Existential Quantifier:**
¬∃x φ ≡ ∀x ¬φ

The statement "nothing is φ" is equivalent to "everything is not φ."

### 2.4.2 Scope and Variable Renaming

Quantifiers bind their variables within their scope. Bound variables can be consistently renamed, provided the renaming does not create conflicts with other variables in the formula.

**Example:** The formula ∀x P(x) is logically equivalent to ∀y P(y). The variable name is arbitrary; what matters is the structure of quantification.

### 2.4.3 Distributivity

Quantifiers distribute over conjunction and disjunction in specific ways:

**Universal Quantifier over Conjunction:**
∀x (φ ∧ ψ) ≡ (∀x φ) ∧ (∀x ψ)

**Existential Quantifier over Disjunction:**
∃x (φ ∨ ψ) ≡ (∃x φ) ∨ (∃x ψ)

Note carefully the asymmetries:
- ∃x (φ ∧ ψ) does NOT generally equal (∃x φ) ∧ (∃x ψ)
- ∀x (φ ∨ ψ) does NOT generally equal (∀x φ) ∨ (∀x ψ)

### 2.4.4 Vacuous Quantification

When a variable does not occur free in a formula, the quantifier is vacuous:

∀x φ ≡ φ (if x is not free in φ)
∃x φ ≡ φ (if x is not free in φ)

## 2.5 Categorical Syllogisms

### 2.5.1 The Traditional Forms

Aristotelian categorical logic deals with statements about categories using four standard forms:

**A (Universal Affirmative):** All S are P. Symbolically: ∀x (S(x) → P(x))

**E (Universal Negative):** No S are P. Symbolically: ∀x (S(x) → ¬P(x))

**I (Particular Affirmative):** Some S are P. Symbolically: ∃x (S(x) ∧ P(x))

**O (Particular Negative):** Some S are not P. Symbolically: ∃x (S(x) ∧ ¬P(x))

### 2.5.2 The Square of Opposition

These four categorical forms stand in systematic logical relations:

- **Contradictories:** A and O are contradictories; E and I are contradictories. One is true and the other false.
- **Contraries:** A and E are contraries; they cannot both be true but can both be false.
- **Subcontraries:** I and O are subcontraries; they cannot both be false but can both be true.
- **Subalternation:** A implies I; E implies O. If the universal is true, the particular is true.

### 2.5.3 Syllogistic Reasoning

A categorical syllogism consists of two premises and a conclusion, each a categorical statement, involving exactly three terms (subject, predicate, and middle term).

**Example:**
Premise 1: All M are P. (Major premise)
Premise 2: All S are M. (Minor premise)
Conclusion: All S are P.

This is a valid syllogistic form (AAA-1 in traditional terminology). In modern notation:

∀x (M(x) → P(x))
∀x (S(x) → M(x))
∴ ∀x (S(x) → P(x))

The validity follows from hypothetical syllogism applied to the corresponding conditionals.

## 2.6 Natural Deduction for Predicate Logic

### 2.6.1 Universal Introduction

The rule of Universal Introduction (∀-Intro) allows the inference of a universally quantified formula:

From a derivation of φ where the variable x is arbitrary (not free in any undischarged assumption), infer ∀x φ.

The requirement that x be arbitrary ensures that the property established holds for all objects in the domain.

### 2.6.2 Universal Elimination

The rule of Universal Elimination (∀-Elim) allows the removal of the universal quantifier:

From ∀x φ, infer φ[t/x], where t is any term and φ[t/x] is the result of substituting t for all free occurrences of x in φ.

**Example:** From ∀x (P(x) → Q(x)) and P(a), we can infer Q(a).

### 2.6.3 Existential Introduction

The rule of Existential Introduction (∃-Intro) allows the introduction of an existential quantifier:

From φ[t/x], infer ∃x φ, provided t is a term that occurs in φ[t/x].

### 2.6.4 Existential Elimination

The rule of Existential Elimination (∃-Elim) allows reasoning from an existential assumption:

From ∃x φ and an assumption φ[y/x] leading to a conclusion ψ (where y is fresh—does not occur in φ, ψ, or any undischarged assumption other than the temporary assumption), infer ψ.

This rule formalizes the reasoning: "There exists something with property φ. Call this arbitrary object 'y' and reason about it. Since we have no special information about y beyond φ, any conclusion reached must hold generally."

## 2.7 Proof Example

Prove: ∀x (P(x) → Q(x)), ∃x P(x) ⊢ ∃x Q(x)

```
1. ∀x (P(x) → Q(x))        Premise
2. ∃x P(x)                 Premise
3. {assume} P(a)           Assumption (where a is arbitrary)
4. P(a) → Q(a)             ∀-Elimination from 1
5. Q(a)                    →-Elimination from 3, 4
6. ∃x Q(x)                 ∃-Introduction from 5
7. ∃x Q(x)                 ∃-Elimination from 2, 3-6
```

The proof demonstrates that if everything with property P also has property Q, and there exists something with property P, then there exists something with property Q.

## 2.8 Limitations and Extensions

### 2.8.1 First-Order Limitations

First-order logic cannot quantify over properties, sets, or relations. For example, the statement "There is a property that all objects share" cannot be expressed in first-order logic.

### 2.8.2 Second-Order Logic

Second-order logic extends first-order logic by allowing quantification over predicates. This enables the expression of statements like "There exists a property P such that..." However, second-order logic lacks the compactness and completeness properties of first-order logic.

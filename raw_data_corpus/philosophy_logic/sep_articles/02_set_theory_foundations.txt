# Set Theory: The Foundation of Mathematical Ontology

## 1. Introduction: The Concept of Set

Set theory provides the foundational framework upon which the entire edifice of modern mathematics is constructed. A set is a collection of distinct objects considered as a whole. The objects comprising a set are called its elements or members. The study of sets and their properties constitutes the most general theory of mathematical collection, providing the underlying logical structure for virtually every branch of formal mathematics.

The significance of set theory extends beyond mere classification. It furnishes the formal apparatus for discussing collections, aggregations, and structural relationships in the most abstract and general terms. Through set theory, mathematicians articulate the concepts of infinity, continuity, and computational decidability that underpin contemporary formal thought.

## 2. Primitive Notions and Fundamental Relations

### 2.1 The Membership Relation

The primitive relation in set theory is the membership relation, denoted by the symbol "∈". The expression "x ∈ A" asserts that the entity x is an element of the set A. This relation is fundamental and irreducible—it cannot be defined in terms of more primitive concepts within the theory itself. All other set-theoretic concepts are constructed from this primitive relation.

The negation of membership is expressed as "x ∉ A," meaning that x is not an element of A.

### 2.2 Set Equality

Two sets are equal if and only if they possess exactly the same elements. This principle, known as the axiom of extensionality, provides the criterion for set identity:

**Axiom of Extensionality:** ∀A ∀B [∀x (x ∈ A ↔ x ∈ B) → A = B]

This axiom establishes that a set is determined entirely by its members. The order of elements and the number of occurrences are irrelevant to set identity. The set {a, b, c} is identical to the set {c, a, b}.

### 2.3 The Empty Set

The empty set, denoted by ∅ or {}, is the unique set that contains no elements. Its existence is guaranteed by the axiom of null set. The empty set serves as the foundational building block from which all other sets are constructed through iterative operations.

The empty set possesses the following properties:
- ∀x (x ∉ ∅)
- ∅ ⊆ A for any set A
- |∅| = 0 (the cardinality of the empty set is zero)

## 3. The Axioms of Zermelo-Fraenkel Set Theory

Modern set theory is founded upon the Zermelo-Fraenkel axioms, commonly abbreviated as ZF. When supplemented with the Axiom of Choice, the system is denoted ZFC. These axioms provide the logical foundation for all set-theoretic constructions.

### 3.1 The Axiom of Extensionality

As stated above, this axiom establishes that sets with identical elements are identical. It provides the identity conditions for sets and ensures that the membership relation determines set identity.

### 3.2 The Axiom of Null Set

There exists a set with no elements. This axiom guarantees the existence of at least one set—the empty set—from which all other sets can be built through subsequent operations.

### 3.3 The Axiom of Pair

Given any two sets A and B, there exists a set {A, B} containing exactly A and B as its elements. From this axiom, the existence of singleton sets {A} follows as a special case. The pair axiom enables the construction of sets from previously existing sets.

### 3.4 The Axiom of Power Set

For every set A, there exists a power set P(A) containing all subsets of A. If A has cardinality n, then P(A) has cardinality 2ⁿ. The power set operation is fundamental to the analysis of infinity and the construction of the real number system.

The power set axiom enables the following construction: Given a set S, the set of all subsets of S, denoted P(S), exists as a well-defined mathematical object.

### 3.5 The Axiom of Union

For every set A, there exists a set ∪A (the union of A) whose elements are exactly the elements of the elements of A. More concretely, if A is a set of sets, the axiom of union guarantees the existence of a set containing all elements of all sets in A.

The union operation enables the combination of multiple sets into a single aggregate set.

### 3.6 The Axiom of Infinity

There exists an infinite set. This axiom is essential for the construction of the natural numbers within set theory. In its standard formulation, it asserts the existence of a set Z such that:
- ∅ ∈ Z
- If A ∈ Z, then A ∪ {A} ∈ Z

The set Z, constructed in this manner, serves as the von Neumann construction of the natural numbers, where each natural number is identified with the set of all smaller natural numbers.

### 3.7 The Axiom Schema of Separation

For any set A and any definable property φ, there exists a subset of A containing exactly those elements of A that satisfy φ. This axiom schema provides the mechanism for defining subsets through property specification.

Formally: ∀A ∃B ∀x (x ∈ B ↔ (x ∈ A ∧ φ(x)))

The separation schema does not create new sets arbitrarily but rather extracts subsets from existing sets.

### 3.8 The Axiom Schema of Replacement

If a definable function F is defined on a set A, then the image F[A] = {F(x) : x ∈ A} is also a set. This axiom schema ensures that the image of a set under a definable function is itself a set.

### 3.9 The Axiom of Foundation

Every non-empty set A contains an ∈-minimal element—an element x such that no element of A is a member of x. This axiom prevents infinite descending chains of the membership relation and ensures that the membership relation is well-founded.

### 3.10 The Axiom of Choice (AC)

For any set A of pairwise-disjoint non-empty sets, there exists a set B (a choice set) that contains exactly one element from each set in A. The Axiom of Choice has been the subject of extensive philosophical debate due to its non-constructive nature—it guarantees the existence of a choice set without providing a method for its construction.

## 4. Operations on Sets

### 4.1 Union

The union of sets A and B, denoted A ∪ B, is the set containing all elements that belong to A or B or both:

A ∪ B = {x : x ∈ A ∨ x ∈ B}

Union is associative and commutative: (A ∪ B) ∪ C = A ∪ (B ∪ C) and A ∪ B = B ∪ A.

### 4.2 Intersection

The intersection of sets A and B, denoted A ∩ B, is the set containing all elements common to both A and B:

A ∩ B = {x : x ∈ A ∧ x ∈ B}

When A ∩ B = ∅, the sets A and B are said to be disjoint.

### 4.3 Difference

The relative complement or difference of A and B, denoted A \ B, is the set of elements in A that are not in B:

A \ B = {x : x ∈ A ∧ x ∈ B}

### 4.4 Cartesian Product

The Cartesian product of A and B, denoted A × B, is the set of all ordered pairs (a, b) where a ∈ A and b ∈ B:

A × B = {(a, b) : a ∈ A ∧ b ∈ B}

The ordered pair (a, b) can be defined set-theoretically as {{a}, {a, b}}, ensuring that (a, b) = (c, d) if and only if a = c and b = d.

## 5. Relations and Functions

### 5.1 Relations

A relation R is a set of ordered pairs. The domain of R is the set of all first components; the range is the set of all second components. A relation R from A to B is a subset of A × B.

A relation R on a set A is:
- **Reflexive:** ∀x ∈ A, (x, x) ∈ R
- **Symmetric:** ∀x, y ∈ A, (x, y) ∈ R → (y, x) ∈ R
- **Transitive:** ∀x, y, z ∈ A, ((x, y) ∈ R ∧ (y, z) ∈ R) → (x, z) ∈ R

An equivalence relation is a relation that is reflexive, symmetric, and transitive.

### 5.2 Functions

A function f from A to B, denoted f: A → B, is a relation such that every element of A is related to exactly one element of B. For each a ∈ A, there exists a unique b ∈ B such that (a, b) ∈ f.

The set of all functions from A to B is denoted B^A, with |B^A| = |B|^|A|.

### 5.3 Injection, Surjection, and Bijection

A function f: A → B is:
- **Injective (one-to-one):** ∀x, y ∈ A, f(x) = f(y) → x = y
- **Surjective (onto):** ∀b ∈ B, ∃a ∈ A such that f(a) = b
- **Bijective:** Both injective and surjective

A bijection establishes a one-to-one correspondence between sets, demonstrating that they have the same cardinality.

## 6. Cardinality and Infinity

### 6.1 Finite Cardinality

Two sets A and B have the same cardinality (|A| = |B|) if there exists a bijection f: A → B. A set is finite if it has the same cardinality as some natural number n, where n is identified with the set {0, 1, 2, ..., n-1}.

### 6.2 Countable Infinity

A set is countably infinite if it has the same cardinality as the natural numbers ℕ. The set ℕ itself, the set of integers ℤ, and the set of rational numbers ℚ are all countably infinite. There exists a bijection between any two countably infinite sets.

### 6.3 Uncountable Infinity

The set of real numbers ℝ is uncountable—it has greater cardinality than ℕ. Cantor's diagonal argument demonstrates that the power set of ℕ (which can be placed in bijection with ℝ) has strictly greater cardinality than ℕ itself.

If |ℕ| = ℵ₀ (aleph-null), then |P(ℕ)| = 2^ℵ₀, which is strictly greater.

### 6.4 The Continuum Hypothesis

The Continuum Hypothesis (CH) asserts that there is no set whose cardinality is strictly between ℵ₀ and 2^ℵ₀. That is, the cardinality of the continuum is ℵ₁. The CH is independent of ZFC—it can neither be proved nor disproved within the standard axiomatic system.

## 7. Ordinal Numbers

Ordinal numbers extend the natural numbers to represent order types of well-ordered sets. The finite ordinals are identified with the natural numbers: 0 = ∅, 1 = {0}, 2 = {0, 1}, and so forth.

The first infinite ordinal is ω = {0, 1, 2, ...}, the order type of the natural numbers. Ordinals are defined as transitive sets well-ordered by ∈.

### 7.1 Ordinal Arithmetic

Ordinal addition is non-commutative. For example, ω + 1 ≠ 1 + ω. The ordinal ω + 1 represents the order type of the natural numbers followed by one additional element, while 1 + ω is isomorphic to ω itself.

Multiplication and exponentiation are similarly non-commutative in the ordinal context, providing a rich structure for analyzing transfinite recursion.

## 8. The Cumulative Hierarchy and the Universe

The cumulative hierarchy V is defined transfinite-recursively:

- V₀ = ∅
- Vₙ₊₁ = P(Vₙ)
- V_λ = ∪_{β<λ} V_β for limit ordinals λ

The universe V of all sets is the union of all Vₙ for ordinals n. Every set appears at some stage in this hierarchy. The cumulative hierarchy provides a well-founded universe of all sets, organized by their construction stage.

## 9. Set Theory as a Foundation

Set theory serves as the foundational framework for all of mathematics. Every mathematical object—numbers, functions, spaces, relations—can be represented as a set. This reduction provides a unified framework in which mathematical statements can be formalized and relationships between different mathematical theories can be precisely analyzed.

The ZFC axioms provide sufficient power to construct virtually all mathematical objects used in contemporary formal reasoning. The consistency of ZFC itself, however, cannot be established within ZFC, as demonstrated by Gödel's incompleteness theorems.
